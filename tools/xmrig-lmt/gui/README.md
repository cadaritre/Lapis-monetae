## LMT Stratum Bridge GUI (Python)

Simple Tkinter UI to configure and run the LMT Stratum bridge.

### Run
```bash
python app.py
```

### Notes
- Set the bridge binary path to your built `lmt-stratum-bridge`.
- Optionally set the XMRig fork binary to auto-start mining.
- The GUI saves settings in `config.json` next to `app.py`.
